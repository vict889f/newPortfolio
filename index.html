My projects

Here are some of the projects i have worked on through KEA.

OWMI - Eksamens project

On this project my group and I worked with the upcoming band OWMI, to create a multimedia solution to promote their brand and help them be seen online. The main focus for the website was to display “the feeling of OWMI”.


Using Wordpress as a database we pulled the data via WP REST API, so that the band could easily add or remove content on the site, and have full creative control.

View website

Kogacenter København

On this project we worked with a Danish reseller of the popular bike brand Koga. Our task was to build a website for the shop owner, and make it as easy as possible for him to add new bikes.


We worked with Wordpress as our backend pulling the data via WP REST API, so the shop owner easily could add or remove bikes from the website.
